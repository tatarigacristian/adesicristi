## Provocari intampinate

### 1. Versiune Node.js incompatibila
- **Problema:** Next.js 16 cere Node >= 20.9.0, dar sistemul avea v18.14.2
- **Solutie:** `nvm use 20` (v20.20.0 era deja instalat)

### 2. Timeout la npm install
- **Problema:** `create-next-app` a dat `ERR_SOCKET_TIMEOUT` la prima incercare
- **Solutie:** Retry dupa switch la Node 20

### 3. Prompt interactiv blocat
- **Problema:** `create-next-app` s-a blocat la "Would you like to use React Compiler?"
- **Solutie:** Pipe `<<< "no"` la stdin

### 4. Port ocupat (EADDRINUSE :3010)
- **Problema:** Dev server-ul nu pornea, portul era ocupat de o instanta anterioara
- **Solutie:** `lsof -ti:3010 | xargs kill -9`

### 5. npm install in directorul gresit
- **Problema:** Pachete instalate in root in loc de `backend/`
- **Solutie:** Verificat cu `ls` si reinstalat in directorul corect

### 6. Redesign major - layout mismatch
- **Problema:** Implementarea initiala (Faza 3) folosea sectiuni verticale fullscreen cu snap. Userul a furnizat HTML-ul Glass template care arata un split layout (sidebar fix + continut scrollabil)
- **Solutie:** Refacere completa a layout-ului: componente separate Sidebar (desktop) si MobileNav (mobile), CSS media queries la breakpoint 1024px
- **Impact:** Cea mai mare provocare - a necesitat rescrierea layout-ului si regandirea componentei Countdown (varianta sidebar compacta + varianta section)
